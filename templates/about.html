About ConnecTTing School

Purpose

The aim of the ConnecTTing Schools web application is to provide users with a simple and convenient tool for deriving network solutions to provide schools thoughout Trinidad and Tobago with sustainable and affordable access to the Internet.

Features

This application features an interactive map to view schools, their configuration status, and the recommended connectivity parameters.

Design Consideration and Assumptions

All users should be fully aware that this application makes some assumptions due to a lack of available information. Users should be guided accordingly. 1) The reference cellular coverage area is based on the locations of cell tower data and does not reflect actual coverage. 2) All numerical values provided were assumed and should be replaced with values more relevant to your circumstance.

Project Motivation

This project is notivated from the work done by the Giga Project, a global inititive which seeks to provide Internet access to all schools around the world.
